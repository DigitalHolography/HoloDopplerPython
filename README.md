# Getting Started

Follow the steps below to install dependencies and run the example.

## 1. Install `uv` if not already intalled

```bash
python -m pip install uv
```

## 2. Sync Dependencies

```bash
uv sync
```

## 3. Install the Project in Editable Mode

```bash
uv pip install -e .
```

## 4. Run the Example

```bash
uv run python -m holodoppler.example
```
